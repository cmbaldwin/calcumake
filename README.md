# CalcuMake

Rails 8.0 3D print project management with invoicing, cost tracking, and pricing calculations.

## Features

- Multi-plate 3D print job calculations
- Invoice generation with auto-numbering
- Multi-currency support (7 languages)
- PWA (Progressive Web App) ready
- User authentication with Devise

## Setup

```bash
bin/setup
bin/dev
```

## Commands

- `bin/dev` - Start development server
- `bin/rails test` - Run tests
- `bin/rubocop` - Code style check
- `bin/brakeman` - Security scan

## Tech Stack

- Rails 8.0 with importmap (no Node.js)
- PostgreSQL
- Bootstrap 5
- Stimulus controllers
- Turbo for SPA-like behavior
